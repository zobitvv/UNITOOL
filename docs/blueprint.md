# **App Name**: Toolbox AI

## Core Features:

- Tool Directory: Categorize and display tools from TinyWow, with the ability to browse by category (PDF, Image, Write, Video, File) based on data from https://tinywow.com.
- Search Functionality: Implement a search bar to quickly find specific tools based on keywords.
- AI-Powered Tool Recommendation: Use AI to suggest the most relevant tool to users based on their input and task, with a LLM acting as a tool selector. The description of each TinyWow tool is provided as a tool to the LLM, when suggesting a particular tool for use.
- TinyWow Redirection: Link directly to the tool functionality on TinyWow while hiding the header and footer of tinywow.com for a seamless experience.
- User Tool List: Allow users to save favourite tools to a personal list for quick access.
- Category Browsing: Allow users to view the tools available in a selected category.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) - A vibrant blue (#3399FF) to convey trust and efficiency, fitting for a tool-oriented site.
- Background color: HSL(210, 20%, 95%) - A light blue-tinted off-white (#F0F8FF) for a clean, modern backdrop that doesn't distract from the tools.
- Accent color: HSL(240, 50%, 60%) - A deeper blue-purple (#735CDD) as a contrast for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif (modern, machined, objective, neutral), because it will appear professional and readable.  The same font is suitable for headlines.
- Use a consistent style of simple, flat icons to represent each tool category (PDF, Image, Write, Video, File).
- Employ a grid-based layout with clear sections for tool categories and search functionality.
- Subtle transitions and animations on hover to provide feedback and improve user experience.
